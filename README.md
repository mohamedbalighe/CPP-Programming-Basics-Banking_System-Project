# Bank Management System (C++)
A professional C++ project developed during my early programming journey to master core concepts.

## Role: Team Leader
I led a dedicated team of **4 members**, managing the project lifecycle, coordinating tasks, and ensuring the integration of individual modules into a cohesive system.

## Project Overview
This application simulates a banking environment to practice **Object-Oriented Programming (OOP)** and complex logic handling.

## 🛠 Key Features
* **Account Control:** Create, modify, and delete accounts.
* **Transaction Engine:** Secure deposits and withdrawals with validation logic.
* **Security:** Basic authentication for user and admin roles.
* **Architecture:** Applied encapsulation and inheritance for clean code.
